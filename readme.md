# 相关说明

用于代码压缩实验。

- index_alibaba_cloud.js: 阿里云基准测试/Layer优化入口文件
- index_compress_alibaba_cloud.js：阿里云代码压缩优化入口文件
- index_tencent_cloud.js: 腾讯云基准测试/Layer优化入口文件
- index_compress_tencent_cloud.js：腾讯云代码压缩优化入口文件
